Quick math
Pour Cost Calculator
Cost/oz → cost/pour → suggested price.
What this means
• Cost per ounce = bottle cost ÷ ounces in bottle
• Cost per pour = cost per ounce × pour size
• Suggested price = cost per pour ÷ (target % / 100)
You can use this to compare a $60 bottle vs $20 bottle instantly.
